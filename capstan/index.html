No surprises

Preview every plan before it writes a byte

Scaffolding tools that write first and explain later are how projects start with mystery files. Capstan inverts it: the plan is the default output, writing is the opt-in. The same philosophy as the rest of the PressGang fleet — explicit over magic, nothing smuggled aboard.
